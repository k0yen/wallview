# WallView

Wallpaper carousel for my collection.

## Features
* **Auto-Index:** GitHub Actions crawls `/wallpapers` and updates the gallery automatically.
* **Minimal:** Dark theme, glassmorphism, and dynamic blurred backgrounds.
* **Navigation:** Use **Arrow Keys** to browse.
* **Download:** One-click save for any image.

## Setup
1. **GitHub Pages:** Enable in `Settings > Pages`.
2. **Permissions:** Set `Settings > Actions > General > Workflow permissions` to **Read and write**.
3. **Usage:** Just drop images into `/wallpapers`. The bot handles the rest.
